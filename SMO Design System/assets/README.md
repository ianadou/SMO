# SMO Brand Assets

- `smo-wordmark.svg` — the SMO wordmark. Geist 700, off-white `#F5F6F8`, with a single blue dot (`#2080FF`) accent positioned after the "O". The dot is the only ornament — it reads as a ball / kickoff beat. Use at 28–32px tall in product chrome (header, login).

When a final logo file lands from the team, replace this file in place — components reference `assets/smo-wordmark.svg` directly.
